# bookworm_adv_anagram_solver
Bookworm Adventure anagram solver using OpenCV and Easy-OCR.
The ROI (Region of Interest) is grabbed from the top left corner of the word box, to the bottom right corner of the word box. 
To detect the ROI, I'm using FeatureMatching from OpenCV. 
After the program detects the ROI successfully, I can press ctrl + alt + d to see the solved anagram.
